# Kavisha Parikh Portfolio

## Setup

1. Install Node.js (v16+)
2. Run `npm install`
3. Create a `.env` file and add your API Key if using Gemini features:
   ```
   VITE_API_KEY=your_key_here
   ```
   (Note: You'll need to update the code to use import.meta.env.VITE_API_KEY instead of process.env.API_KEY)
4. Run `npm run dev`

## Enjoy!
